# Async Await Photos App (TypeScript)

A simple frontend project demonstrating **TypeScript**, **async/await**, and **API data fetching** using the JSONPlaceholder Photos API.  
Photos are displayed in a responsive grid using **Bootstrap 5**.

---

## Features

- Written in **TypeScript** with strict type safety
- Uses **async/await** for API calls
- Fetches data from a public REST API
- Responsive UI with **Bootstrap**
- Clean separation of model, service, and UI logic

---

## Demo API

Data is fetched from:

https://jsonplaceholder.typicode.com/photos
